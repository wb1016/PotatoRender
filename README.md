# PotatoRenderer
a mod that removes ambient occulusion and adds resolution adjustment feature.
this can achieve higher FPS with visual tradeoffs.
supports both OpenGL and Vulkan renderer.
## porting
i will not port this onto neoforge, forge, older versions, whatever.
# source
some sources are from [RenderScale](https://github.com/Zolo101/RenderScale) mod. affected files are marked with code comments.
